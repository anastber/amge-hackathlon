# E-commerce Delivery Time Prediction

## Overview
This project is part of AMGE-Caravane's first Data & AI Hackathon, organized by the Association des Marocains aux Grandes Écoles (5,000+ members from French engineering and business schools).

## Goal
Predict delivery time in days for e-commerce orders using only features available at order placement, before any shipment occurs.

## Results
Achieved a score of **2.84884 error days**, ranking **3rd out of 5 teams**.

## Main Files
- **`full_pipeline.ipynb`** - Complete analysis and modeling pipeline
- **`Final_presentation.pptx`** - Final presentation slides

The project focuses on building the best predictive model while extracting business insights for strategy development.
